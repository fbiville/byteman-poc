# Simple Byteman P.O.C.

## Bytemanless

Needless to say you need a JDK. 

 1. run `compile.sh`
 1. run `runWithoutByteman.sh`

## Byteman!

 1. set up [Byteman](http://www.jboss.org/byteman/)
 1. run `compile.sh`
 1. run `runWithByteman.sh`
